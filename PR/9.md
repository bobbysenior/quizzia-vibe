## 8. Page `/my-quizzes` (Mes quiz)

**Fichiers créés :**
- `src/app/my-quizzes/page.tsx` — Server Component (requiert auth)
- `src/app/my-quizzes/quiz-list-client.tsx` — Client Component avec actions

**Fonctionnalités :**
- Affiche **tous les quiz créés par l'utilisateur**
- Organisés en **3 sections distinctes** :
  - **Brouillons** (dot gris)
  - **Publiés** (dot vert)
  - **Archivés** (dot orange)
- Chaque section affiche le **compteur** de quiz
- Chaque quiz affiche un **tag de statut** (Brouillon / Publié / Archivé)

**Boutons par statut :**

| Statut | Boutons disponibles |
|--------|---------------------|
| Brouillon | Modifier · **Publier** · Supprimer |
| Publié | Modifier · **Archiver** · Supprimer |
| Archivé | Modifier · **Republier** · Supprimer |

- **Modifier** : redirige vers `/quizzes/[id]/edit`
- **Publier** : passe le statut à `published`
- **Archiver** : passe le statut à `archived`
- **Republier** : passe le statut de `archived` à `published`
- **Supprimer** : suppression définitive avec `confirm()`

## 9. Server Actions pour la gestion des statuts

**Fichier modifié :**

**Actions ajoutées :**
- `publishQuizAction(quizId)` → statut `published`

**Fichier modifié :**

**Fonctions ajoutées :**
- `updateQuizStatus(quizId, status)` — met à jour le statut avec vérification `creator_id`
- `deleteQuiz(quizId)` — supprime le quiz avec vérification `creator_id`
