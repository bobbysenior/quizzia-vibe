# PR #7 — Génération de quiz par IA (Cerebras)

## Résumé

L'utilisateur décrit un quiz, le backend appelle Cerebras (`gpt-oss-120b`) qui retourne un JSON structuré, validé par Zod puis inséré en base.

## Fichiers

| Fichier | Rôle |
|---|---|
| `src/lib/ai/cerebras.ts` | Client Cerebras singleton |
| `src/lib/ai/generate-quiz.ts` | Schéma Zod, system prompt, appel chat completion (`response_format: json_object`) |
| `src/lib/ai/create-quiz-from-llm.ts` | Insertion batch quiz → questions → choix (service role key) |
| `src/app/api/quizzes/generate/route.ts` | `POST /api/quizzes/generate`, auth par cookies Supabase |
| `src/app/quizzes/new/prompt-form.tsx` | Formulaire prompt + min/max questions, redirection vers l'éditeur |
| `scripts/test-cerebras.mjs` | `npm run test:api` — sanity-check de la clé et du modèle |

## Format JSON attendu du LLM

```json
{
  "title": "...",
  "theme": "...",
  "questions": [
    {
      "question_text": "...",
      "choices": [
        { "choice_text": "...", "is_correct": true }
      ]
    }
  ]
}
```

## Endpoint

```
POST /api/quizzes/generate
Content-Type: application/json
Cookie: <session Supabase>

{ "prompt": "...", "min_questions": 5, "max_questions": 10 }

→ 201 { "quizId": "uuid" }
```

## Variables d'environnement

```
CEREBRAS_API_KEY=csk-...
# CEREBRAS_MODEL=gpt-oss-120b (optionnel)
```
