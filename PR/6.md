# PR #6 — Pages de jeu (jouer à un quizz)

## Résumé

Ajout des 4 pages nécessaires pour jouer à un quizz de bout en bout : lister les quizz, voir les détails, répondre question par question, voir son score final.

---

## Pages créées

| Route | Fichier | Type |
|---|---|---|
| `/quizzes` | `page.tsx` | Server Component |
| `/quizzes/[id]` | `page.tsx` | Server Component |
| `/quizzes/[id]/play` | `page.tsx` | Client Component |
| `/quizzes/[id]/results` | `page.tsx` | Server Component |

---

## Fonctions service ajoutées (`quizzes.service.ts`)

| Fonction | Description |
|---|---|
| `getQuizQuestions(quizId)` | Retourne les questions avec leurs choix (sans `is_correct`) |
| `createQuizAttempt(quizId, total)` | Crée une tentative (anonyme ou connecté) |
| `getExistingAttempt(attemptId)` | Récupère une tentative existante |
| `submitAnswer(attemptId, questionId, choiceId)` | Vérifie la réponse, insère `user_answers`, retourne feedback |
| `completeAttempt(attemptId)` | Calcule le score, marque la tentative comme terminée |

---

## Server Actions (`quizzes/[id]/play/actions.ts`)

```
startAttempt(quizId, total)  → crée l'attempt + récupère les questions
answerQuestion(attemptId, qId, choiceId) → soumet une réponse
finishAttempt(attemptId)     → finalise la tentative
```

---

## Flux de jeu

1. **`/quizzes`** — Grille de quizz publiés (design cohérent avec la landing)
2. **`/quizzes/[id]`** — Fiche détail : titre, thème, nombre de questions, bouton "Commencer"
3. **`/quizzes/[id]/play`** — Questions une par une :
   - Barre de progression
   - 4 choix par question
   - Feedback immédiat (✓ Bonne réponse / ✗ Pas tout à fait)
   - Animation de transition entre questions
   - Redirection automatique vers les résultats après la dernière
4. **`/quizzes/[id]/results`** — Score final :
   - Pourcentage en grand + barre de progression
   - Message personnalisé selon le score
   - Boutons Rejouer / Voir tous les quiz

---

## Accessible aux anonymes

Toute la boucle de jeu fonctionne sans compte. L'`attempt.user_id` est `NULL` pour les anonymes.
