# PR — feature/quiz-page → main

Document consolidé des modifications de la branche `feature/quiz-page`.

---

## Vue d'ensemble

Implémentation complète du système de création et d'édition de quiz :

1. Page `/quizzes/new` avec deux modes : création manuelle et génération par IA (OpenAI puis Cerebras)
2. Formulaire manuel : titre, thème, nombre de questions → création d'un quiz brouillon avec questions/choix vides
3. Page `/quizzes/[id]/edit` : édition locale de toutes les questions et choix, sauvegarde groupée en base
4. Génération IA : appel OpenAI/Cerebras, parsing JSON structuré, insertion batch
5. Correctifs d'authentification (cookies Next.js 16, proxy.ts)

## Flux de création manuelle

```
ManualForm → createQuizAction (Zod: 5-30)
  → createQuiz() : INSERT quizzes (draft)
  → createEmptyQuestions(N) : INSERT batch questions ("Nouvelle question") + 4 choix ("Choix A".."D")
  → redirect /quizzes/[id]/edit
```

## Flux d'édition

```
Édition locale (React state) :
  - Titre, thème, questions, choix → tout modifiable
  - Suppression/ajout de questions et choix
  - Max 4 choix par question
  - "Marquer correct" en local

[Bouton Enregistrer] → saveFullQuizAction :
  1. UPDATE quizzes (title, theme, question_count, updated_at)
  2. DELETE FROM questions WHERE quiz_id (CASCADE choices)
  3. INSERT batch questions
  4. INSERT batch choices
```

## Flux IA

```
PromptForm → POST /api/quizzes/generate (Bearer token)
  → Vérifie token → appelle Cerebras/OpenAI
  → LLM répond JSON { title, theme, questions: [{question_text, choices}] }
  → Validation Zod → create-quiz-from-llm : INSERT quizzes + questions + choices
  → redirect /quizzes/:id/edit
```
