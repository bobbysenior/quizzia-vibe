# PR #5 — Pages Login, Stats et refonte Landing

## Résumé

Création des pages login et stats connectées à l'API, refonte de la landing page selon la maquette Quizia, mise en place du design system.

---

## Pages créées / modifiées

| Fichier | Description |
|---|---|
| `src/app/page.tsx` | Landing refaite : hero aurora, preview, marquee, how it works, bibliothèque live, CTA banner |
| `src/app/login/page.tsx` | Split panel (brand + form), toggle Connexion/Inscription, lié à signIn/signUp |
| `src/app/stats/page.tsx` | Dashboard protégé : 4 KPI, historique, mes quiz — données réelles Supabase |
| `src/app/layout.tsx` | Nav + footer globaux, récupération user pour Nav |
| `src/app/globals.css` | Design tokens de la maquette en directives Tailwind v4 |
| `src/components/nav.tsx` | Navigation sticky avec backdrop blur, liens conditionnels, signOut |
| `src/lib/services/quizzes.service.ts` | Ajout `getUserStats()`, `getUserAttempts()`, `getUserQuizzes()` |

---

## Connecté à l'API

- **Login** : `signIn()` / `signUp()` via `useActionState`, validation Zod, états loading/error
- **Landing** : `listQuizzes()` → grille de quizz dynamique avec gradients
- **Stats** : `getUserStats()` (KPI), `getUserAttempts()` (historique), `getUserQuizzes()` (mes quiz)
  - Route protégée : redirection `/login` si non authentifié

---

## Design system (repris de `maquette/`)

- Palette : bg `#fbfbfd`, ink `#1d1d1f`, accent `oklch(58% 0.22 280)`
- Polices : Geist Sans + Geist Mono
- Composants : cards, tags, buttons, forms, nav, footer
- Animations : rise, shine, blur backdrop
